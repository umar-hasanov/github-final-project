# Simple Interest Calculator

This project is a simple interest calculator implemented using a Bash script.

## Description
The simple interest calculator computes the simple interest based on three user inputs:
- Principal amount
- Rate of interest
- Time period

## Formula
The simple interest is calculated using the formula:

Simple Interest = (Principal × Rate × Time) / 100

## Usage
Run the script in a terminal and provide the required inputs when prompted.

```bash
./simple-interest.sh
